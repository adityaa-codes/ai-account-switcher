# v0.1.0 Release Notes

**Release date:** 2026-02-18

## Highlights

`ai-account-switcher` v0.1.0 is the first public release of a unified profile manager for Gemini CLI and Codex CLI on Linux. It provides account switching, credential import/export, health checks, and installer-driven shell integration for day-to-day multi-account workflows.

## What's Included

### Core CLI

- Unified command surface for both providers:
  - `switcher gemini ...`
  - `switcher codex ...`
- Profile operations: `add`, `list`, `switch`, `next`, `remove`, `import`, `export`
- Global commands: `status`, `config`, `install`, `uninstall`, `version`

### Authentication & Profile Management

- Gemini support:
  - OAuth profiles (`oauth_creds.json` + keyring compatibility)
  - API key profiles
- Codex support:
  - API key profiles
  - ChatGPT OAuth-style `auth.json` profiles
- Atomic symlink-based activation for active credentials
- XDG-compliant storage under `~/.config/ai-account-switcher/`

### Health Checks

- `switcher gemini health` and `switcher codex health`
- Refresh/API validation across supported auth types
- Status persisted per profile (`valid`, `expiring`, `expired`, `revoked`, `unknown`)

### Automation & Integration

- Installer for shell integration and environment loading
- Gemini hook integration:
  - BeforeAgent pre-check support
  - AfterAgent quota-exhaustion auto-rotation support
- Gemini slash command integration:
  - `/change` mapped to `switcher gemini next`

## Developer Tooling

- Packaging via `hatchling`
- Python 3.10+ target
- Ruff linting/formatting and mypy strict typing configured
- Pytest test harness and coverage tooling configured

## Notes

- Codex custom slash commands are not installed in v0.1.0.
- Existing OAuth tokens may require re-authentication depending on provider/account state.
